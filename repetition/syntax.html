Syntax

Es gibt drei verschiedene Arten html mit CSS zu ergänzen und somit zu Stylen.

Inline CSS, Internal CSS und External CSS.



Diese zwei Zeilen können mit allen drei Varianten erstellt werden:

Das ist eine grüne Headline

Das ist ein blauer Paragraph



Inline CSS

        
        <h1 style="color:rgb(5, 86, 34);">Das ist eine grüne Headline</h1>
        <p style="color:rgb(8, 0, 255);">Das ist ein blauer Paragraph</p>

Internal CSS

        
    <!DOCTYPE html>
    <html>
    <head>
    <style>
    h1   {rgb(5, 86, 34);}
    p    {rgb(8, 0, 255);}
    </style>
    </head>
    <body>
    
    <h1>Das ist eine grüne Headline</h1>
    <p>Das ist ein blauer Paragraph</p>
    
    </body>
    </html>

External CSS

Beim External CSS wird ein zusätzliches CSS-File erstellt. Dieses nennt sich styles.css


    

    <!DOCTYPE html>
    <html>
    <head>
    <link rel="stylesheet" href="styles.css">
    </head>
    <body>

    <h1>Das ist eine grüne Headline</h1>
    <p>Das ist ein blauer Paragraph</p>

    </body>
    </html>
